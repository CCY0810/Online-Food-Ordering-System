# 🍔 Online Food Ordering System

A web-based food ordering platform that enables customers to browse menus, place orders, and track order status through an interactive ordering interface.

The system is designed to streamline restaurant operations by supporting **three user roles: Admin, Staff, and Customer**, each with dedicated functionalities to manage menu items, customer orders, and operational workflows efficiently.

---

# ✨ Key Features

## 👤 Role-Based User System
The platform supports three types of users:

### Customer
- Register and login to the system
- Browse menu items with categorized listings
- Search food items
- Add items to shopping cart
- Place food orders
- Track order status
- Provide feedback and ratings

### Staff
- View incoming orders
- Update order status
- Manage customer order workflow
- Communicate with customers regarding order updates

### Admin
- Manage menu categories
- Add, update, and delete food items
- Manage user accounts
- Monitor customer orders
- Manage inventory stock
- View sales reports and analytics

---

# 🛒 Ordering Workflow

1. Customer browses menu items
2. Customer adds items to cart
3. Customer places an order
4. Staff processes and updates order status
5. Customer tracks order progress

# 🎯 Project Objective

The goal of this project is to demonstrate the development of a **dynamic web application with role-based access control**, database integration, and real-time order management features.

The system showcases how web technologies can be used to build an interactive platform that improves efficiency in food ordering and restaurant management.
